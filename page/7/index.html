vscode 使用手册

2018-10-22

界面

  • 开关右侧小地图 : ctrl+shift+p -> toggle minimap
  • 开关左侧侧边栏 : ctrl+b
  • 关闭窗口 : ctrl+w

编辑

  • 选中单词/选中下一个相同单词 : ctrl+d
  • 选中光标处相同单词 : ctrl+shift+l
  • 选中一行 : ctrl+i
  • 删除当前行 : shift+del
